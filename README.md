# Tara Anand Portfolio Website

A modern, responsive portfolio website for Tara Anand - Brand Strategist & Content Creator.

## Features

- **Modern Design**: Dark theme with vibrant orange accents matching the Figma design
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Fade-in animations and smooth scrolling
- **Interactive Navigation**: Active link highlighting based on scroll position
- **Contact Form**: Functional contact form with validation
- **Performance Optimized**: Lazy loading images and optimized assets

## Sections

1. **Hero Section**: Introduction with large name display and tagline
2. **Stats Section**: Key metrics and achievements
3. **About Section**: Personal introduction and professional background
4. **Brands Section**: Showcase of brands worked with
5. **Services Section**: Services offered (Content Strategy, Creative Direction, etc.)
6. **Work Section**: Portfolio case studies
7. **Testimonials Section**: Client testimonials
8. **Contact Section**: Contact form and discovery call information

## Getting Started

1. Open `index.html` in a web browser
2. The website will load with all styles and functionality

## Image Assets

**Note**: The website currently uses image URLs from the Figma MCP server (`http://localhost:3845/assets/...`). These will only work if you have the Figma MCP server running locally.

To use your own images:
1. Replace the image URLs in `index.html` with your hosted image URLs
2. Or download the images from Figma and place them in an `assets` folder, then update the paths

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --color-orange: #ff6a00;
    --color-black: #000000;
    /* ... other colors */
}
```

### Content
Edit the HTML content in `index.html` to update text, links, and sections.

### Fonts
The website uses Google Fonts. To change fonts, update the font-family declarations in `styles.css`.
